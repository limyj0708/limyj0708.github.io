Asynchronous, Synchronous, Blocking, Non-Blocking

Nov 11, 2019 in Computer Science

л§ҺмқҖ мһҗлЈҢмқҳ мӮ° мӨ‘м—җм„ң, к°ҖмһҘ м•Ңкё° мүҪкі  м§ҒкҙҖм ҒмңјлЎң м„ӨлӘ…н•ң мһҗлЈҢлҠ”
https://stackoverflow.com/questions/2625493/asynchronous-vs-non-blocking
мқҙ м§Ҳл¬ёмқҳ м„ё лІҲм§ё лӢөліҖмқҙлқјлҠ” кІ°лЎ мқ„ лӮҙл ёлӢӨ.
мқҙ лӢөліҖмқҳ лІҲм—ӯ + ліҙм¶© м„ӨлӘ…мқ„ мң„н•ң лӢӨлҘё мһҗлЈҢл“Ө + мӮ¬мЎұмқ„ м„һм–ҙм„ң кёҖмқ„ м Ғм–ҙліҙм•ҳлӢӨ.

synchronous / asynchronous : л‘җ лӘЁл“Ҳ мӮ¬мқҙмқҳ кҙҖкі„м—җ лҢҖн•ң н‘ңнҳ„
blocking / non-blocking : лӘЁл“Ҳ н•ҳлӮҳмқҳ мғҒнғңм—җ лҢҖн•ң н‘ңнҳ„

мҳҲлҘј л“Өм–ҙ,
лӘЁл“Ҳ X : лӮҳ
лӘЁнҠӨ Y : м„ңм җ
Xк°Җ Yм—җкІҢ м§Ҳл¬ё : C++ primer мұ… мһҲлӮҳмҡ”?

1. Blocking

  • Yк°Җ Xм—җкІҢ лӢөн•ҳкё° м „к№Ңм§Җ, XлҠ” кё°лӢӨлҰ°лӢӨ. XлҠ” Blocking мғҒнғңм—җ л№ м§„ кІғмқҙлӢӨ.

2. Non-Blocking

  • Yк°Җ Xм—җкІҢ лӢөн•ҳкё° м „м—җ, XлҠ” лӢӨлҘё мқјмқ„ н•  мҲҳ мһҲлӢӨ.
  • Xк°Җ 2л¶„л§ҲлӢӨ Yк°Җ мқјмқ„ лҒқлғҲлҠ”м§Җ нҷ•мқён• к№Ң?(Synchronousлқјл©ҙ мқҙл ҮкІҢ лҗ  кІғ к°ҷлӢӨ.) м•„лӢҲл©ҙ Yк°Җ лӢӨ лҗҗлӢӨкі  л¶ҖлҘҙл©ҙ нҷ•мқён• к№Ң?(Asynchronousлқјл©ҙ мқҙл ҮкІҢ лҗ  кІғ к°ҷлӢӨ.) лӘЁлҘёлӢӨ.(= мғҒкҙҖмқҙ м—ҶлӢӨ.)
  • мҡ°лҰ¬к°Җ м•„лҠ” кұҙ Xк°Җ Yк°Җ мқјмқ„ лҒқлӮҙкё° м „м—җ лӢӨлҘё мқјмқ„ н•  мҲҳ мһҲлӢӨлҠ” кІғ лҝҗмқҙлӢӨ. XлҠ” Non-BlockingмқҙлӢӨ.

3. Synchronous

  • Yк°Җ Xм—җкІҢ лӢөн•ҳкё° м „м—җлҠ”, XлҠ” лӢӨлҘё мқјмқ„ м§„н–үн•ҳм§Җ м•ҠлҠ”лӢӨ - лқјкі  м„ӨлӘ…н•ҳкі  мһҲлҠ”лҚ°, мқҙлҹ¬л©ҙ Blockingкіј м •мқҳк°Җ к°ҷлӢӨ. мўҖ лӢӨлҘҙкІҢ мғқк°Ғн•ҙ ліҙмһҗ.
  • Synchronousм—җлҠ” мӨ‘мҡ”н•ң л‘җ к°Җм§Җ нӮӨмӣҢл“ңк°Җ мһҲлӢӨ.
    • мһ‘м—…мқҳ мҲңм„ңлҘј л§һм¶”лҠ” кІғ#Thread_or_process_synchronization). мҷң мҲңм„ңлҘј л§һм¶”лғҗкі ? м—¬лҹ¬ мһ‘м—…мқҙ лҸҷмӢңм—җ(Concurrently) Critical sectionм—җ м§„мһ…н•ҳлҠ”кІғмқ„ л§үкё° мң„н•ҙм„ңмқҙкё°лҸ„ н•ҳкі , нҠ№м • мҲңм„ңм—җ л§һкІҢ мһ‘м—…л“Өмқ„ мӢӨн–үн•ҙм•ј н•  н•„мҡ”к°Җ мһҲкё°(н‘ңлҘј мӮ¬м§ҖлҸ„ м•Ҡкі  л№„н–үкё°м—җ нғҲ мҲҳлҠ” м—ҶлӢӨ) л•Ңл¬ёмқҙкё°лҸ„ н•ҳлӢӨ. мһ‘м—… мҲңм„ңлҘј л§һм¶ң л•Ң BlockingмңјлЎң мІҳлҰ¬н•ҳл©ҙ нҺён•ҳкё° л•Ңл¬ём—җ Blockingмқҳ к°ңл…җмқҙ м„һм—¬м„ң л“ұмһҘн•ҳлҠ” кІғ лҝҗмқҙлӢӨ.
      • мһ‘м—… мҲңм„ңмқҳ кҙҖм җм—җм„ң м„ӨлӘ…н•ҳлҠ” кёҖ л‘җ к°ң

      • мўӢмқҖ мҳҲмӢң : мғҒмӮ¬к°Җ мҷҖм„ң м–ҙл–Ө мқјмқ„ мІҳлҰ¬н•ҳлқјкі  л§җн•ңлӢӨ. к·ёлҰ¬кі  лӮҙ л“ұ л’Өм—җм„ң мӢңмІҙлҘј л…ёлҰ¬лҠ” лҸ…мҲҳлҰ¬л§ҲлғҘ лӮҳлҘј міҗлӢӨліҙкі  мһҲлӢӨ. вҖңмһҗл„Өк°Җ мқјмқ„ лӢӨ лҒқлӮј л•Ңк№Ңм§Җ м—¬кё°м„ң кё°лӢӨлҰҙ кұём„ё.вҖқ
      • мғҒнғң нҷ•мқёмқҳ кҙҖм җм—җм„ң м„ӨлӘ…н•ҳлҠ” кёҖ : https://homoefficio.github.io/2017/02/19/Blocking-NonBlocking-Synchronous-Asynchronous/
  • мқҙ X,Y мҳҲм—җм„ңлҠ”, SynchronousлҠ” Xк°Җ Yм—җкІҢ мұ… м°ҫм•ҳлғҗкі  л¬јм–ҙліҙкі ,(callerк°Җ calleeмқҳ мғҒнғң нҷ•мқё) Yк°Җ мұ…мқҙ мһҲлҠ”м§Җ м—ҶлҠ”м§Җ Xм—җкІҢ м•Ңл Ө мӨҖ мқҙнӣ„м—җм•ј, Xк°Җ Yм—җкІҢ вҖңк·ёлһҳм„ң мқҙ мұ… м–јл§ҲмЈ ?вҖқ лқјкі  л¬јм–ҙліҙлҚҳк°Җ, вҖңк·ё мұ… мЈјл¬ё мўҖ н•ҙмЈјм„ёмҡ”вҖқ лқјкі  мҡ”мІӯн•  мҲҳ мһҲлҠ” мғҒнҷ©мқё кІғмқҙлӢӨ.(мһ‘м—… мҲңм„ң) мұ…мқҙ мһҲлҠ”м§Җ нҷ•мқён•ң лӢӨмқҢм—җ к°ҖкІ©мқ„ л¬јм–ҙліҙлҚҳм§Җ мЈјл¬ёмқ„ мҡ”мІӯн•ҳлҚҳм§Җ н•  мҲҳ мһҲмңјлӢҲк№Ң. Xк°Җ Yк°Җ мұ… м°ҫлҠ” лҸҷм•Ҳ лӢӨлҘё л¬ҙм–ёк°ҖлҘј н•ҳлҠ” кұҙ мғҒкҙҖмқҙ м—ҶлӢӨ. м„ңм җ м•һм—җм„ң мӨ„л„ҳкё°лҘј н•  мҲҳлҸ„ мһҲм§Җ м•ҠмқҖк°Җ? мұ…м—җ кҙҖл Ёлҗң лӢӨмқҢ мқјмқ„ лӘ»н•ҳлҠ” кІғ лҝҗ.

  • мқҙл ҮкІҢ лҙҗлҸ„ мғҒлӢ№нһҲ н—·к°ҲлҰ¬кё° л•Ңл¬ём—җ, мҪ”л“ңлҘј ліҙмһҗ. м•„лһҳмқҳ мҪ”л“ңлҠ” Synchronous & Non-Blocking н•ң к°„лӢЁн•ң мҪ”л“ңмқҙлӢӨ.

    • Xк°Җ 2мҙҲл§ҲлӢӨ Yк°Җ лӢөмқ„ мӨ¬лҠ”м§Җ м•„лӢҢм§Җ нҷ•мқён•ңлӢӨ.
    • Yк°Җ кІ°кіјлҘј л°ҳнҷҳн•ҳкё° мң„н•ҙ мӨҖл№„ мӨ‘мқҙм–ҙлҸ„, Xмқҳ while лЈЁн”„лҠ” кі„мҶҚ лҸҢм•„к°Җкі  к·ё м•Ҳм—җм„ң лӢӨлҘё мһ‘м—…мқ„ м§„н–үн•  мҲҳ мһҲлӢӨ. к·ёлһҳм„ң Non-BlockingмқҙлӢӨ.
    • н•ҳм§Җл§Ң, whileл¬ёмқ„ л№ м ёлӮҳк°Җм„ң лӢӨлҘё мһ‘м—…мқ„ н•  мҲҳлҠ” м—ҶлӢӨ. к·ёлһҳм„ң SynchronousлӢӨ.
    • мҪ”л“ңмқҳ мҳҲмӢңк№Ңм§Җ ліҙл©ҙ, Xк°Җ м„ңм җмқ„ л– лӮҳм§Җ лӘ»н•ңлӢӨкі  н•ҙм„қн•  мҲҳ мһҲлӢӨ. Yк°Җ мұ…мқ„ м°ҫм•„мӨҳм„ң м„ңм җкіј кҙҖл Ёлҗң мқјмқ„ л§Ҳм№ҳкё° м „м—җлҠ”, м„ңм җмқ„ л– лӮҳм„ң лӢӨлҘё кұё н•  мҲҳ м—ҶлӢӨ. м„ңм җ м•һм—җм„ң мӨ„л„ҳкё°лҠ” к°ҖлҠҘн•ҙлҸ„.(whileл¬ё м•Ҳм—җм„ң лӯ”к°Җ лӢӨлҘё мһ‘м—…)
    • Blockingмқҙм—ҲлӢӨл©ҙ, XлҠ” м•„л¬ҙкІғлҸ„ лӘ»н•ҳкі  кё°лӢӨл Өм•ј н–ҲкІ м§Җл§Ң.

4. Asychronous

  • Yк°Җ Xм—җкІҢ лӢөн•ҳкё° м „м—җ, XлҠ” лӢӨлҘё кіім—җ к°Җм„ң лӢӨлҘё мқјмқ„ н•  мҲҳ мһҲлӢӨ. XлҠ” Yк°Җ л¶ҖлҘҙкё° м „к№Ңм§Җ лҸҢм•„мҳӨм§Җ м•ҠлҠ”лӢӨ. мқҙ л•Ң XмҷҖ YлҠ” Asychronous н•ҳлӢӨкі  л§җн•ңлӢӨ.
  • м—¬кё°м„ңлҸ„ SynchronousмҷҖ к°ҷмқҖ л‘җ к°Җм§Җ нӮӨмӣҢл“ңлЎң мӮҙнҺҙліҙмһҗ.
    • мһ‘м—…мқҳ мҲңм„ңк°Җ ліҙмһҘлҗҳм§Җ м•ҠмқҢ : AsynchronousлҠ” м—„л°ҖнһҲ л§җн•ҳл©ҙ, мһ‘м—…л“Өмқҙ кіөнҶөм ҒмңјлЎң мӮ¬мҡ©н•ҳлҠ” global clockмқҙ м—Ҷкі , мӢ нҳёлӮҳ л©”м„ём§Җмқҳ лҸ„м°© мӢңк°„мқҙ мһ‘м—…мқҳ мӢ лў°м„ұм—җ мҳҒн–Ҙмқ„ лҜём№ҳм§Җ м•ҠмқҢмқ„ лң»н•ңлӢӨ. мҰү мһ‘м—…мқҳ мҲңм„ңк°Җ ліҙмһҘлҗҳм§Җ м•ҠлҠ”лӢӨ.(A,B,C мҲңм„ңлЎң мӢӨн–үлҗҳм—ҲмңјлӮҳ мҷ„лЈҢлҸ„ A,B,C мҲңм„ңмқј кІғмқҙлқј ліҙмһҘн•  мҲҳ м—ҶмқҢ)
      • мӢ нҳёлӮҳ л©”м„ём§Җмқҳ лҸ„м°© мӢңк°„мқҙ мһ‘м—…мқҳ мӢ лў°м„ұм—җ мҳҒн–Ҙмқ„ лҜём№ҳм§Җ м•ҠмқҢ, мҰү, к°Ғ мһ‘м—…мқҙ м„ңлЎң м—°кҙҖлҗҳм§Җ м•Ҡм•„м„ң л¶„лҰ¬лҗ  мҲҳ мһҲмңјл©°, мһ‘м—… м§Җм—°мӢңк°„мқҙ нҒ° кІҪмҡ°м—җ мһҳ нҷңмҡ©лҗ  мҲҳ мһҲлӢӨ. (DB м ‘к·ј, Http мҡ”мІӯ, File I/O л“ұ)
    • Calleeк°Җ мһҗмӢ мқҳ мҷ„лЈҢ мғҒнғңлҘј нҷ•мқён•ҳл©°, callbackмңјлЎң Callerм—җкІҢ мһҗмӢ мқҳ мҷ„лЈҢлҘј м•ҢлҰј
      • к·јлҚ° мӮ¬мӢӨ мҷ„лЈҢ нҶөліҙлҘј н•ҙлҸ„ лҗҳкі  м•Ҳн•ҙлҸ„ лҗңлӢӨ. мҷ„лЈҢ нҶөліҙк°Җ callerм—җкІҢ мқҳлҜёк°Җ мһҲлҠҗлғҗ м—ҶлҠҗлғҗмқҳ л¬ём ңмқҙлӢӨ.(лӮҙк°Җ н•ң м§Ҳл¬ёмқҙлӢӨ :D)
      • мўӢмқҖ мҳҲмӢң : мғҒмӮ¬к°Җ мҷҖм„ң м–ҙл–Ө мқјмқ„ мІҳлҰ¬н•ҳлқјкі  л§җн•ңлӢӨ. к·ёлҰ¬кі  лӢӨлҘё мқј н•ҳлҹ¬ к°ҖлІ„лҰј. мқјмқ„ лӢӨ лҒқлӮҙл©ҙ, лӮҳлҠ” мғҒмӮ¬м—җкІҢ вҖңлӮҳ лӢӨн•Ё!вҖқ мқҙлқјкі  л§җн•ңлӢӨ.
  • Yм—җкІҢ мұ… мһҲлғҗкі  л¬јм–ҙліё нӣ„м—җ Xк°Җ м№ҙнҺҳм—җ к°Җм„ң м»Өн”јлҘј л§ҲмӢңкё° мӢңмһ‘н–Ҳм§Җл§Ң, мұ… м°ҫкё°ліҙлӢӨ м»Өн”ј л§ҲмӢңкё°к°Җ лҚ” л№ЁлҰ¬ лҒқлӮ  мҲҳлҸ„ мһҲлӢӨ. мқҙ л‘ҳмқҖ мҷ„м „нһҲ лі„к°ңмқҳ мһ‘м—…мқҙл©°, мһ‘м—…мқҳ мҲңм„ңк°Җ ліҙмһҘлҗҳм§Җ м•ҠлҠ”лӢӨ.(Asynchronous & Non-Blockingмқҙлқјл©ҙ.) к·ёлҰ¬кі  XлҠ” Yк°Җ XлҘј л¶ҖлҘј л•Ң м„ңм җмңјлЎң лҸҢм•„к°„лӢӨ.

к°Ғ 2к°ңм”©мқҳ к°ңл…җмқҙ мһҲмңјлӢҲ, мҙқ 4к°ңмқҳ мЎ°н•©мқҙ лӮҳмҳ¬ кІғмқҙлӢӨ.
